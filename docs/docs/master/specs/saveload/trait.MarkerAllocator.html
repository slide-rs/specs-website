This allocator is used with the Marker trait. It provides method for allocation new Markers. It should also provide a Marker -> Entity mapping. The maintain method can be implemented for cleanup and actualization. See docs for Marker for an example.

Required Methods

fn allocate(&mut self, entity: Entity, id: Option<M::Identifier>) -> M

Allocate new Marker. Stores mapping Marker -> Entity. If id argument is Some(id) then the new marker will have this id. Otherwise allocator creates marker with a new unique id.
